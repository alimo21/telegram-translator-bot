# 🤖 ربات ترجمه تلگرام | عربی ← فارسی

ربات خودکار کپی و ترجمه پست‌های کانال عربی به فارسی

## ویژگی‌ها

- ✅ کپی خودکار تمام پست‌ها (متن، عکس، فیلم، گیف، فایل، صدا)
- ✅ ترجمه خودکار عربی به فارسی
- ✅ بدون نیاز به ادمین بودن در کانال مبدا
- ✅ ارسال بدون ذکر منبع (دقیقاً مثل پست اصلی)
- ✅ داشبورد وب مدیریتی
- ✅ لاگ کامل عملکرد

## نحوه نصب و راه‌اندازی

### پیش‌نیازها

1. **API ID و API Hash** از [my.telegram.org](https://my.telegram.org)
2. **توکن ربات** از [@BotFather](https://t.me/BotFather)
3. یک **کانال تلگرام** که ربات در آن ادمین باشد

### دیپلوی در Railway

1. این پروژه را در گیتهاب آپلود کنید
2. در [railway.app](https://railway.app) یک پروژه جدید بسازید
3. **Deploy from GitHub** را انتخاب کنید
4. یک دیتابیس **PostgreSQL** اضافه کنید
5. متغیر `DATABASE_URL` خودکار تنظیم می‌شود
6. آدرس وب اپلیکیشن را باز کنید و تنظیمات را وارد کنید

### مراحل تنظیم

1. **تب تنظیمات**: توکن ربات، API ID، API Hash، آدرس کانال‌ها را وارد کنید
2. **تب احراز هویت**: با شماره تلفن تلگرام وارد شوید
3. **تب کنترل ربات**: دکمه شروع را بزنید

## توضیحات فنی

- **MTProto Protocol**: برای خواندن پست‌های کانال مبدا (بدون نیاز به ادمین بودن)
- **Bot API**: برای ارسال پست‌ها به کانال مقصد
- **Google Translate API**: برای ترجمه خودکار عربی به فارسی
- **PostgreSQL + Drizzle ORM**: برای ذخیره تنظیمات و جلوگیری از ارسال تکراری
- **Next.js**: فریمورک وب برای داشبورد مدیریت
